# 🚀 BTC 回测系统 - 立即开始

## 一键启动

```bash
# 方法 1: 使用启动脚本
./start_backtest.sh

# 方法 2: 直接运行 API
python3 backend/backtest/api.py
```

## 访问系统

启动后，浏览器会自动打开，或手动访问：

**Web UI**: http://127.0.0.1:8001/backtest.html

## 快速测试（5 步）

### 1. 选择策略模板
在页面顶部的"策略模板"下拉菜单中选择 **"EMA金叉策略"**

### 2. 设置回测日期
- 开始日期：选择 3 个月前
- 结束日期：选择今天

### 3. 点击运行
点击 **"▶ 运行回测"** 按钮

### 4. 等待完成
等待几秒钟，系统会自动运行回测

### 5. 查看结果
- 📊 性能指标：总收益、胜率、最大回撤、夏普比率
- 📈 权益曲线图
- 📝 详细交易记录

## 创建自己的策略

### 开仓条件示例
```
条件 1: EMA7 > EMA25  （EMA7 上穿 EMA25）
条件 2: RSI14 < 70    （RSI 未超买）
逻辑: AND             （两个条件都满足）
```

### 平仓条件示例
```
条件 1: EMA7 < EMA25  （EMA7 下穿 EMA25）
逻辑: OR              （任一条件满足）
```

### 止盈止损
```
止盈: 10%  （盈利 10% 时自动平仓）
止损: 5%   （亏损 5% 时自动平仓）
```

## 可用指标

### 趋势指标
- **EMA7, EMA12, EMA25, EMA50**: 指数移动平均线

### 动量指标
- **RSI6, RSI14**: 相对强弱指标
- **MACD DIF, DEA, MACD**: MACD 指标

### 波动率指标
- **布林上轨, 布林中轨, 布林下轨**: 布林带
- **ATR**: 平均真实波幅

### 价格和成交量
- **收盘价 (close)**
- **成交量 (volume)**

## 运算符

- `>` : 大于
- `<` : 小于
- `>=` : 大于等于
- `<=` : 小于等于
- `==` : 等于
- `!=` : 不等于

## 常见问题

### Q: 回测没有交易？
**A**: 可能是条件太严格，尝试：
- 放宽开仓条件
- 扩大回测日期范围
- 减小持仓大小

### Q: 数据库连接失败？
**A**: 检查 MySQL 是否运行：
```bash
mysql.server status
mysql.server start
```

### Q: 端口被占用？
**A**: 更换端口或杀死占用进程：
```bash
lsof -i :8001
kill -9 <PID>
```
